# Big-Data-and-IKhiIAS

##### Big Data
* Lab1 - создание клиента для HDFS
* Lab2 - алгоритмы кросс-корреляции на MapReduce
* Lab3 - Hive, Pig
* Lab4 - алгоритм HITS для ранжирования страниц
* Lab5 - Apache Spark
